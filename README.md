# 📊 Table to Markdown Converter

WebUIでテーブルを編集し、リアルタイムでMarkdown形式に変換するアプリケーション

## 🎯 目的

Markdownで直接テーブルを書くのは面倒です。このアプリを使えば、視覚的にテーブルを編集し、即座にMarkdown形式を取得できます。

## ✨ 機能

### 基本機能
- **直感的な編集**: セル内のテキストを直接編集可能
- **リアルタイム変換**: 編集内容が即座にMarkdownに反映
- **行・列の操作**: 各行・列の横にある ➕ ➖ ボタンで自由に追加・削除
- **ワンクリックコピー**: 生成されたMarkdownをクリップボードにコピー
- **レスポンシブデザイン**: デスクトップとモバイルに対応

### 高度な機能
- **セルのドラッグ&ドロップ**: セル右上の `⋮⋮` をドラッグしてセル同士を入れ替え
- **文字寄せ設定**: ヘッダー行の左右端にある薄い線をクリックして左寄せ・右寄せを設定
- **キーボードショートカット**:
  - `Tab` / `Shift+Tab`: セル間を左右に移動
  - `↑` `↓` `←` `→`: セル間を上下左右に移動

## 📝 操作方法

### 基本操作
1. **セルの編集**: セルをクリックして直接テキストを入力
2. **行を追加**: 各行の左側にある ➕ ボタンをクリックして、その行の下に追加
3. **列を追加**: 各列の上にある ➕ ボタンをクリックして、その列の右に追加
4. **行を削除**: 各行の左側にある ➖ ボタンをクリックしてその行を削除
5. **列を削除**: 各列の上にある ➖ ボタンをクリックしてその列を削除
6. **テーブルをクリア**: 上部の「Clear All」ボタンですべての内容をクリア
7. **Markdownをコピー**: 「Copy」ボタンでクリップボードにコピー

### 高度な操作
- **セルの入れ替え**: セルにホバーすると右上に表示される `⋮⋮` アイコンをドラッグして、別のセルにドロップすると内容が入れ替わります
- **文字寄せの変更**: ヘッダー行のセルにホバーすると左右に薄い線が表示されます。左側をクリックすると左寄せ、右側をクリックすると右寄せになります
- **キーボードナビゲーション**: セル編集中に `Tab` キーで右へ、`Shift+Tab` で左へ、矢印キーで上下左右に移動できます
