# 📊 Table to Markdown Converter

WebUIでテーブルを編集し、リアルタイムでMarkdown形式に変換するアプリケーション

## 🎯 目的

Markdownで直接テーブルを書くのは面倒です。このアプリを使えば、視覚的にテーブルを編集し、即座にMarkdown形式を取得できます。

## ✨ 機能

- **直感的な編集**: セル内のテキストを直接編集可能
- **リアルタイム変換**: 編集内容が即座にMarkdownに反映
- **行・列の操作**: 簡単に行や列を追加・削除
- **ワンクリックコピー**: 生成されたMarkdownをクリップボードにコピー
- **レスポンシブデザイン**: デスクトップとモバイルに対応

## 🚀 使い方

### オンラインで使用

GitHub Pagesにデプロイされています：
https://isamuJazz.github.io/table2markdown/

### ローカルで使用

1. リポジトリをクローン
```bash
git clone https://github.com/isamuJazz/table2markdown.git
cd table2markdown
```

2. ブラウザでindex.htmlを開く

## 📝 操作方法

1. **セルの編集**: セルをクリックして直接テキストを入力
2. **行を追加**: 「➕ 行を追加」ボタンをクリック
3. **列を追加**: 「➕ 列を追加」ボタンをクリック
4. **行を削除**: 「➖ 行を削除」ボタンで最後の行を削除
5. **列を削除**: 「➖ 列を削除」ボタンで最後の列を削除
6. **クリア**: 「🗑️ クリア」ボタンでテーブル内容をクリア
7. **コピー**: 「📋 コピー」ボタンでMarkdownをクリップボードにコピー
