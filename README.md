# 📊 Table to Markdown Converter

WebUIでテーブルを編集し、リアルタイムでMarkdown形式に変換するアプリケーション

## 🎯 目的

Markdownで直接テーブルを書くのは面倒です。このアプリを使えば、視覚的にテーブルを編集し、即座にMarkdown形式を取得できます。

## ✨ 機能

### 基本機能
- **直感的な編集**: セル内のテキストを直接編集可能
- **リアルタイム変換**: 編集内容が即座にMarkdownに反映
- **行・列の操作**: 各行・列の横にある ➕ ➖ ボタンで自由に追加・削除
  - ➕ ボタン：現在位置に挿入し、既存の行・列を下/右に移動
  - ➖ ボタン：その行・列を削除
- **双方向編集**: MarkdownエリアでMarkdownを直接編集し、「反映」ボタンでテーブルに反映
- **ワンクリックコピー**: 生成されたMarkdownをクリップボードにコピー
- **レスポンシブデザイン**: デスクトップとモバイルに対応

### Vim風モーダル編集
このアプリケーションはVimエディタに着想を得た3つのモードを持ちます：

#### ノーマルモード
- セルの移動とドラッグ&ドロップが可能
- **移動**: `h` `j` `k` `l` または矢印キー、`Tab` / `Shift+Tab`
- **操作**:
  - `i`: 編集モードへ
  - `v`: ビジュアルモードへ
  - `d`: セルの内容を削除
  - `x`: セルの内容をカット（クリップボードに保存）
  - `y`: セルの内容をヤンク（コピー）
  - `p`: クリップボードの内容をペースト
  - `u`: 直前の操作をアンドゥ
- **ドラッグ&ドロップ**: セルをドラッグして入れ替え

#### 編集モード
- セル内のテキストを自由に編集
- `Esc`: ノーマルモードへ戻る

#### ビジュアルモード
- 矩形範囲を選択してバッチ操作
- **選択拡張**: `h` `j` `k` `l` または矢印キー
- **操作**:
  - `d`: 選択範囲を削除
  - `x`: 選択範囲をカット
  - `y`: 選択範囲をヤンク
  - `p`: 選択範囲にペースト
- `Esc`: ノーマルモードへ戻る

### その他の高度な機能
- **文字寄せ設定**: ヘッダー上部のボタン（←, ↔, →）で列の配置を変更
- **アンドゥ機能**: `u` キーで直前の操作を取り消し
- **ヘルプモーダル**: 画面上部の `?` アイコンで使い方を確認

## 📝 操作方法

### 基本操作
1. **セルの編集**: セルをクリックして編集モードに入り、直接テキストを入力
2. **行を追加**: 各行の左側にある ➕ ボタンをクリックして、その位置に挿入（既存の行は下に移動）
3. **列を追加**: 各列の上にある ➕ ボタンをクリックして、その位置に挿入（既存の列は右に移動）
4. **行を削除**: 各行の左側にある ➖ ボタンをクリックしてその行を削除
5. **列を削除**: 各列の上にある ➖ ボタンをクリックしてその列を削除
6. **テーブルをクリア**: 上部の「Clear All」ボタンですべての内容をクリア
7. **Markdownをコピー**: 「Copy」ボタンでクリップボードにコピー
8. **Markdownを反映**: Markdownエリアを編集後、「反映」ボタンでテーブルに反映

### Vim風モーダル操作

#### ノーマルモードでの操作
- **移動**:
  - `h` / `←`: 左へ
  - `j` / `↓`: 下へ
  - `k` / `↑`: 上へ
  - `l` / `→`: 右へ
  - `Tab`: 右へ
  - `Shift+Tab`: 左へ
- **編集操作**:
  - `i`: 編集モードへ切り替え
  - `v`: ビジュアルモードへ切り替え
  - `d`: カーソル位置のセル内容を削除
  - `x`: カーソル位置のセル内容をカット（削除してクリップボードに保存）
  - `y`: カーソル位置のセル内容をヤンク（コピー）
  - `p`: クリップボードの内容をカーソル位置にペースト
  - `u`: 直前の操作をアンドゥ
- **ドラッグ&ドロップ**: セルをドラッグして別のセルにドロップすると内容が入れ替わります

#### 編集モードでの操作
- 通常のテキスト入力が可能
- `Esc`: ノーマルモードへ戻る

#### ビジュアルモードでの操作
- **選択範囲の拡張**:
  - `h` `j` `k` `l` または矢印キー: 選択範囲を拡張
- **編集操作**:
  - `d`: 選択範囲のすべてのセルを削除
  - `x`: 選択範囲のすべてのセルをカット
  - `y`: 選択範囲のすべてのセルをヤンク
  - `p`: 選択範囲のすべてのセルにペースト
- `Esc`: ノーマルモードへ戻る

### その他の操作
- **文字寄せの変更**: ヘッダー上部のボタン（←, ↔, →）をクリックして列の配置を変更
- **ヘルプの表示**: 画面上部の `?` アイコンをクリックして使い方を確認
